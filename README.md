
# Image Processing API
is a web service for image processing.

### Table of contents

- Install
- Lint
- Build
- Test
- Run

### Install
    npm install
    
### Linting

 - `npm run format`
 - `npm run lint`

### Test
    npm test

### Build
    npm run build
    
### Run
 - development mode
	 -  `npm run dev `
- production mode
	- `npm run start`

### Check Health
GET / `http://localhost:4000/api` to check for api health .
